# Portfolio_html5_css3
Projeto "Meu Portfolio profissional interativo"

***

## Objetivo 

Criação de um website estático responsivo e interativo Profissional, desenvolvido com HTML5 e CSS3, hospedado na plataforma de nuvem GitHub pages, funcionando como Portfólio digital.

## Introdução:

O Projeto "Meu Portfólio interativo" compreende:

- A apresentação pessoal com uma seção "sobre mim";
- A construção de um Portfólio de projetos (exibição de trabalhos concluídos ou em andamento, com links diretos para seus respectivos repositórios);
- Contenha integração social com links estratégicos e iconografia para Linkedin e GitHub;
- Responsividade em abordagem mobile-firts (Adaptação adequada para diversos tipos de dispositivos);
- Formulário ou canal de contato.


**Como o projeto será desenvolvido?**

Para o funcionamento organizacional mais fluído o trabalho está dividido em três marcos progessivos: 
- Planejamento e esqueleto semântico (wireframe básico);
- Estética e Adaptabilidade (CSS3 e utilização de media queries);
- Lançamento e documentação.


**E por quer construir esse projeto?**

As vantagens da construção desse projeto são diversas, e dentre elas está o aprendizado adquirido na construção de artefatos web, demonstração de conhecimentos técnicos, bem como organização de código com intenção profissional. Além disso, estabelece a base de versionamento e estrutura necessária para os próximos desafios.
